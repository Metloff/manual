# Быстрая сортировка (quick sort)

**Свойства:**
- рекурсивный алгоритм

**Временная сложность:** 
- Худшее время:  `O(N^2)`
- Среднее время: `O(NlogN)`
- Лучшее время: `O(NlogN)`
